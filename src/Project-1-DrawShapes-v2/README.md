# Project #1: DrawShapes v2

This is an enhanced version of the DrawShapes application that allows users to create, manipulate, and persist various shapes.

## Features

### Basic Features
- **Multiple Shapes**: Create squares, circles, and rectangles with precise positioning
- **Interactive Selection**: Right-click to select individual shapes, drag to select multiple shapes
- **Color Options**: Choose from five colors (Red, Blue, Green, Yellow, Orange) for your shapes
- **Visual Feedback**: Selected shapes are highlighted with a darker shade of their color


### Enhanced Features
- **Movement Controls**: Move selected shapes using arrow keys when in Move mode
- **Scaling Operations**: Scale shapes up/down using arrow keys or mouse wheel
- **File Operations**: Save scenes to text files and load them back
- **Undo Functionality**: Revert to previous states with Ctrl+Z


### Advanced Features
- **Layer Management**: Change shape stacking order with Bring to Front/Send to Back
- **Multi-selection**: Drag-select multiple shapes simultaneously
- **Keyboard Shortcuts**: Accelerator keys for common operations


## How to Use

1. Select a shape type from the Shape menu (Square, Circle, Rectangle)
2. Select a color from the Color menu (Red, Blue, Green, Yellow, Orange)
3. Click on the canvas to create a shape
4. Right-click on shapes to select them
5. Use the Operation menu to switch between drawing, moving, and resizing modes
6. Use arrow keys to move or resize selected shapes when in the appropriate mode
7. Use the mouse wheel to quickly scale selected shapes
8. Use File menu to save/load scenes and undo operations
